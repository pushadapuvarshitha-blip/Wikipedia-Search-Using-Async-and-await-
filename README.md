Wikipedia Search Application 
A lightweight, responsive web application that allows users to search for Wikipedia articles in real-time. Built using vanilla web technologies, this project connects directly to the official MediaWiki API to fetch and display relevant articles instantly. 
Features 

• Real-Time Search: Fetches and displays matching Wikipedia articles as you type or submit. 
• Instant Previews: Displays the article title, a short snippet, and a direct link to the full page. 
• Responsive Design: Optimized for seamless viewing on desktops, tablets, and mobile devices. 
• Clean UI: A minimalist, user-friendly interface focused on readability. 

Technologies Used 

• HTML5: Structured the search interface and result layouts. 
• CSS3: Flexbox/Grid layouts, custom typography, and responsive media queries. 
• JavaScript (ES6+): Handles user input, asynchronous API calls (), and dynamic DOM manipulation. 
• Wikipedia API: Utilizes the official MediaWiki Action API for fetching search results.

How It Works 

1. The user types a query into the search bar and clicks the search button or presses . 
2. A JavaScript  request is sent to the Wikipedia API endpoint with the search parameters. 
3. The API returns data in JSON format containing the matching articles. 
4. JavaScript processes the JSON array and dynamically injects the titles, snippets, and URLs into the DOM.
